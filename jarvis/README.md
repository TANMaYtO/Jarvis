# Jarvis AI Assistant

A voice-controlled AI assistant inspired by Iron Man's JARVIS. This assistant helps with everyday tasks such as:

- Opening applications
- Searching the web
- Providing weather information
- Setting reminders
- Managing calendars
- Answering questions

## Features

- Voice recognition and response
- Application control
- Web search capabilities
- Natural language processing
- Weather forecasts
- Reminders and scheduling

## Installation

1. Clone this repository
2. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```
3. Configure your API keys in the `config.py` file
4. Run the assistant:
   ```
   python main.py
   ```

## Requirements

- Python 3.8 or higher
- Microphone for voice input
- Speakers for voice output
- Internet connection for web-based features

## Usage

Simply start the application and speak your commands. Start with "Jarvis" followed by your request:

- "Jarvis, open Chrome"
- "Jarvis, what's the weather today?"
- "Jarvis, search for AI news"
- "Jarvis, set a reminder for 3 PM"
